# Daniel Perez Arandia

Esto fiue desarrollado por Daniel perez con ayuda de Gemini en puntos especificos para optimizar el codigo en funciones de gemini 3, no todo esta hecho con Gemini 3.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
